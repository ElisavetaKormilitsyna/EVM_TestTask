# EVM_TestTask

Считывание показаний с термистров организовано по DMA, отключен непрерываный режим, считывание данных происходит с частотой 10 Гц, как указано в условии задачи. Частота снятия показаний настроена таймером АЦП, запуск происходит аппаратно, программно перезапускать старт DMA не нужно.
Соответсвенно, очередь с данными заполняется с указанной частотой и по мере заполнения обработанные показания отправляются в CAN.

Реализовала вышеописанным образом, так как из условий задачи неясно, нужен ли бОльший поток данных, фильтрация или усреднение показаний. 
В случае необходимости накопления и обработки показаний с датчиков задачу отправки в CAN можно бы было сделать периодической (пробуждать раз в 100мс) или же класть обработанные данные в очередь на отправку раз в 100 мс.
